# 🐍 Система генерации и проверки заданий Python

Автоматическая система для генерации учебных заданий по программированию на Python и проверки решений студентов с использованием нейронных сетей.

## 🎯 Описание проекта

Этот проект представляет собой веб-приложение для автоматической генерации учебных заданий по программированию на Python и проверки решений студентов. Система использует простую нейронную сеть для анализа качества кода и предоставляет детальную обратную связь.

## ✨ Основные возможности

### 🎲 Генерация заданий
- **Автоматическое создание** заданий различной сложности
- **Различные категории**: алгоритмы, обработка данных, функции
- **Пользовательские задания** с возможностью настройки
- **Готовые тестовые случаи** для каждого задания
- **Подсказки и шаблоны** для студентов

### 🔍 Проверка решений
- **Синтаксическая проверка** Python кода
- **Безопасное выполнение** кода в изолированной среде
- **Автоматическое тестирование** с тестовыми случаями
- **Анализ качества кода** с помощью нейронной сети
- **Детальная обратная связь** и рекомендации

### 🧠 Анализ с помощью нейронной сети
- **Оценка правильности** решения
- **Анализ эффективности** алгоритма
- **Оценка читаемости** кода
- **Предложения по улучшению** стиля программирования

## 🚀 Быстрый старт

### Установка зависимостей

```bash
# Клонирование репозитория (если используется Git)
git clone <repository-url>
cd python_task_system

# Установка зависимостей
pip install -r requirements.txt
```

### Запуск приложения

```bash
# Запуск в режиме разработки
python main.py

# Или с переменными окружения
export FLASK_DEBUG=True
export FLASK_HOST=0.0.0.0
export FLASK_PORT=5000
python main.py
```

### Доступ к приложению

После запуска приложение будет доступно по адресу:
- **Локально**: http://127.0.0.1:5000
- **В сети**: http://0.0.0.0:5000

## 🔧 Использование

### Для преподавателей

1. **Создание задания**:
   - Перейдите на страницу "Генерация заданий"
   - Выберите категорию и сложность
   - Нажмите "Сгенерировать задание"
   - Получите готовое задание с тестовыми случаями

2. **Создание кастомного задания**:
   - Выберите "Пользовательское задание"
   - Заполните название и описание
   - Добавьте тестовые случаи в формате JSON
   - Сгенерируйте задание

### Для студентов

1. **Решение задания**:
   - Выберите задание из списка
   - Изучите условие и тестовые случаи
   - Напишите решение в редакторе кода
   - Нажмите "Проверить решение"
   - Получите оценку и рекомендации

## 🛠️ Разработка

### Добавление новых типов заданий

1. Отредактируйте `task_generator.py`
2. Добавьте новый шаблон в `task_templates`
3. Обновите категории заданий

### Расширение анализатора кода

1. Модифицируйте `code_analyzer.py`
2. Добавьте новые признаки для анализа
3. Обновите нейронную сеть для работы с новыми признаками

### Кастомизация интерфейса

1. Измените шаблоны в папке `templates/`
2. Добавьте стили в `static/css/`
3. Расширьте JavaScript в `static/js/`
